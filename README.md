# Helmet and Number Plate Detection using Deep Learning

## 🎯 Project Overview
This project detects whether a motorbike rider is wearing a helmet and identifies the vehicle's number plate using deep learning and computer vision.

## 🧠 Technologies Used
- Python  
- OpenCV  
- YOLOv8 (Ultralytics)  
- PyTorch  
- Tesseract OCR  
- Flask (optional web interface)

## 📁 Folder Structure
helmet_number_plate_detection/
├── data/ # Datasets (images and labels)
│ ├── helmet/
│ └── number_plate/
├── models/ # Trained model weights
├── notebooks/ # Experiments and training notebooks
├── src/ # Python source code
├── web/ # Web interface
├── results/ # Output images, CSVs, logs
└── docs/ # Project documentation
